# 📂 Smart File Organizer

A simple and intelligent **Flask-based web application** that automatically organizes files into categorized folders based on file types. It provides a clean and modern UI to make file management effortless.

---

## 🚀 Features

- 📁 Organize files by type (Images, Documents, Videos, Audio, etc.)
- ⚡ One-click file organization
- 🧠 Automatic file categorization using Python
- 💻 Clean and modern UI (Flask + HTML + CSS)
- 📊 Displays summary (Moved, Failed, Skipped)
- 🔒 Safe and lightweight application

---

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌐
- HTML5
- CSS3
- Jinja2 Templates
